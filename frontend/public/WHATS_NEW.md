# What's New in StockGame v2 🚀

안녕하세요! StockGame이 v2로 대폭 업데이트되었습니다.

## 주요 변경 사항

### 1. 기본 자금 상향 💰
- 기본 자금이 **5억**으로 상향되고, 신용한도 또한 **5억**이 되었습니다.
- 이를통해 보다 더 적극적인 운용이 가능합니다.

### 2. 새로운 매매 방법 추가 ⚖️
- 이제 **공매도**를 통한 매매가 가능합니다. 신용한도를 사용하여 미리 주식을 팔고 나중에 커버할 수 있습니다.
- 또한 **지정가 매매**가 추가되었습니다. 원하는 가격에 매도/매수를 설정해놓으면 해당 가격이 도달했을때 자동으로 매매가 실행됩니다.

### 3. 더 많은 종목, 종목 추가 기능 도입 🔍
- 코스피 100종, 코스닥 100종을 기본으로 미국주식은 ETF를 포함하여 다양한 종목이 추가되었습니다.
- 또한 검색창을 이용해서 원하는 종목을 **검색 후 추가**하여 매매할 수 있습니다.

### 4. 차트 기능 강화 📈
- 이제 차트는 과거 **1년간**의 데이터를 제공합니다.
- 차트에 이동평균선, 거래량 등의 지표를 확인하실 수 있습니다.

### 5. UI/UX 개선 ✨
- 주요 지표 티커(코스피, 코스닥, S&P 500, Dow Jones, NASDAQ 등) 추가
- 리더보드 히스토리 확인기능 (Chart race 애니메이션 추가) 
- 전체 포트폴리오에 롱/숏 구분하여 표시
- 사용자 합산 수익률 상위/하위 5종목 표시
- 여기서 종목 선택시 보유 사용자 정보 확인
- 주식 목록에서 전체/즐겨찾기/보유종목 순으로 토글 가능
- 지정가 매매 목록 (목록에서 취소가능)
- AI 리포트 강화 (일정시간마다 포트폴리오 변화를 감지하여 자동 생성)

### 6. 일일 미션 시스템 🎁
- 매일 자정에 사용자 별로 새로운 미션이 주어집니다.
- 미션을 완료하면 포인트를 획득할 수 있습니다.
- 다만, 포인트는 아직 사용처가 없습니다.
